# Pulse Recovery Test Plan

Thanks for helping test Pulse Recovery. The aim is to check whether the app is easy to use and whether the recovery feedback feels clear and useful.

## 1. Create an account / sign in

Please test:

* Registering a new account
* Signing back in later
* Whether any login or error messages are clear

## 2. Create a recovery assessment manually

After exercise, enter:

* Peak heart rate
* Heart rate after 60 seconds
* Heart rate after 120 seconds
* How hard the workout felt, from 1–10
* How you felt afterwards, from 1–10

Please check whether the input screen is understandable and whether the result makes sense.

## 3. Review the result

After saving an assessment, look at:

* The recovery percentage
* The recovery classification
* The recovery curve/pattern explanation
* The advice or coaching message

Please tell me if anything feels unclear, repetitive, too technical, or misleading.

## 4. Check the history page

After creating more than one assessment, check:

* Whether assessments appear correctly in history
* Whether the charts display properly
* Whether the dates and values look right
* Whether the summaries are easy to understand

## 5. Try a few different scenarios

Useful tests include:

* A hard workout
* An easy workout
* A short recovery test while sitting down
* A session where the heart-rate values barely change
* A session where you feel tired despite normal heart-rate recovery

## 6. Report anything odd

Please send me:

* Screenshots of anything confusing or broken
* The phone model and iOS version
* What you were doing when the issue happened
* Whether the result/advice felt useful or not.

The app is for general fitness and recovery feedback only. It is not intended to diagnose medical conditions or replace medical advice.
